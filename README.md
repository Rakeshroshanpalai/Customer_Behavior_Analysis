# Customer_Behavior_Analysis
Data analytics project showcasing customer behavior and analysis using Python, SQL and Power Bi

Overview
This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI. It covers data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and report generation. The project concludes with insights and a presentation (PPT) created using Gamma.

Dataset
Source: CSV/Excel dataset
Format: Structured tabular data
Purpose: Analyze business data, identify trends, and generate actionable insights.

Project Steps
Load the dataset using Python.
Perform data cleaning (missing values, duplicates, formatting).
Conduct Exploratory Data Analysis (EDA).
Import the cleaned data into SQL.
Write SQL queries to extract business insights.
Connect SQL data to Power BI.
Build interactive dashboards and KPIs.
Create a final report and presentation in Gamma.

How to Run
1. Clone the Repository

cd data-analytics-project
2. Install Required Libraries
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
3. Run the Python Script
python main.py
4. Import Data into SQL
Create a database in PostgreSQL, MySQL, or SQL Server.
Import the cleaned dataset.
Execute the SQL queries provided in the project.
5. Open Power BI
Connect Power BI to your SQL database.
Refresh the data.
Open the .pbix file to explore the dashboard.
